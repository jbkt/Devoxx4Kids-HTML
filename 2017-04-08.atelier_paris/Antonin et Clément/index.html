Commandes minecraft:

Les commandes basiques:

  • /time set [day/night] Pour changer le temps jours/nuit.
  • /weather [clear/rain] Pour changer la météo.
  • /gamemode [0/1/2/3] Pour changer de mode de jeu(0=survie;1=créatif;2=aventure;3=spectateur).
  • /clear [TonPseudo/AutrePseudo/rien] Pour vider son inventaire ou celui des autres.
